BOTSY - Quick Start Guide

Requirements:
- Python 3.8 or higher

1. Create a virtual environment (optional but recommended):
   Linux/Mac:
       python3 -m venv env
       source env/bin/activate
   Windows:
       python -m venv env
       env\Scripts\activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run the program:
   python botsy_2.py

The application will launch in your terminal and guide you through each step.

Note:
- Botsy will launch in Catalan by default.
- To change the interface language, press option 3 from the main menu to access the language selection menu.
